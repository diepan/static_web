Setup

Welcome to your Lavarel 5 example project on Nitrous.

Running the development server:

In the Nitrous IDE, start Lavarel 5 Example via “Run › Start Laravel 5 Example ”
Now you've got a development serer running and can see the output in the Nitrous terminal window. You can open up a new shell or otilize tmux to open new shell to run other commands.

Preview the app

In the Nitrous IDE, open the “Preview” menu and click “Port 8000”.
